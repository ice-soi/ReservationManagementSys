# 予定管理アプリケーション
カレンダー形式の予定入力管理システムです。  

## 開発言語
 - C#

## 概要
 - IBMCloudを使っています。
 - DBはNoSQLDBを使っています。
 - 画面はSPAで構成しています。

## 画面
　1. アクセスすると当日を基準にしたカレンダーを表示します。
